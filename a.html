¿Cómo Funciona el DNS?

Cuando un usuario ingresa una URL en su navegador, el DNS se pone a trabajar para conectar la URL con la dirección IP del servidor actual. Esto es llamado la resolución del nombre de DNS e involucra un recursor DNS poniendo en fila a varios servidores de nombres para descifrar la dirección IP real de un servidor.

¿Qué tipos de registros DNS puedo configurar?

Los tipos de registros DNS que puedes configurar en tu dominio son: A: apunta a una IP. El servidor de destino es el encargado de tratar la solicitud entrante. Hay que tener en cuenta si la IP destino es fija o dinámica. Ejemplo: 192.168.1.104 AAAA: es un tipo de registro parecido al anterior, dado que también apunta a una IP. Se diferencian en que apunta hacia una dirección IP IPv6. Ejemplo: 2001:0dd8:85a3:08d3:1319:8a2e:0370:7334 CNAME: apunta a otro dominio o a otro subdominio. Ejemplo: blog.example.com

¿Qué son los nombres de servidor DNS?

Los nombres de servidor determinan quién manejará las peticiones que se hagan al dominio en cuanto a varios servicios como web, email entre otros. También es parte del Sistema de Nombre de Dominio (DNS por sus siglas en inglés), que mantiene un directorio de los dominios y sus respectivas direcciones IP. Esto significa que cuando un usuario visita un dominio, se hace una búsqueda DNS para verificar el nombre del servidor y la configuración DNS.
